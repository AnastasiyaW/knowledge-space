# SEO Promotion Strategy

A structured plan that defines what SEO actions to perform, in what order, and over what timeline. Built before starting work on any project, projected for 6-12 months. Different site types (e-commerce, services, information) require fundamentally different strategies.

## Key Facts

- Before building a strategy, answer two questions: "What TOP positions do I want in 6 months?" and "What TOP positions do I want in 1+ year?"
- Strategy is built at the project's zero point and projected 6-12 months forward; it should be revisited and adjusted quarterly based on analytics data
- The three pillars of any SEO strategy: **technical foundation** (audit + fix), **content** (semantic core + optimization), **authority** (link building + behavioral)
- For **new sites**: technical setup + structure -> content creation -> link building (in this order; links to thin content are wasted)
- For **existing sites**: technical audit -> quick wins (fix critical errors) -> content gaps -> link building -> ongoing optimization
- Standard project phases: Month 1 = audit + setup, Month 2 = semantic core + content plan, Month 3-4 = content creation + on-page optimization, Month 5+ = link building + expansion
- See [[keyword-research-semantic-core]] for the foundational research that drives strategy
- See [[seo-analytics-tools]] for measuring strategy effectiveness

## Patterns

### Strategy by site type

```
E-COMMERCE (online store):
Priority: category pages first (higher volume), then product pages
Month 1:  Technical audit, structure review, analytics setup
Month 2:  Semantic core for top categories, competitor analysis
Month 3:  Category page optimization (meta tags, text, filters)
Month 4:  Product page templates, image optimization
Month 5+: Link building to category pages, content marketing
Key:      Fix faceted navigation / filter pages early (crawl budget)

SERVICE SITE:
Priority: service pages (transactional), then blog (informational)
Month 1:  Technical audit, structure, analytics
Month 2:  Semantic core for services, local SEO setup
Month 3:  Service page optimization, Google Business Profile
Month 4:  Blog content for informational queries
Month 5+: Local link building, review acquisition
Key:      Local SEO is critical - Google Business Profile optimization

INFORMATION SITE / BLOG:
Priority: High-volume informational clusters
Month 1:  Technical audit, topic research
Month 2:  Content plan based on semantic core
Month 3+: Content production at scale (3-5 articles/week)
Month 6+: Internal linking optimization, content updates
Key:      Volume and freshness matter most; update old content
```

### Competitive analysis framework

```
1. Identify competitors:
   - Search top 5 keywords -> note top-ranking domains
   - Use Ahrefs/SEMrush "Competing Domains" report

2. Analyze competitor metrics:
   - Domain Rating / Domain Authority
   - Total organic traffic estimate
   - Number of ranking keywords
   - Number of referring domains
   - Content volume (total indexed pages)

3. Gap analysis:
   - Keywords they rank for, you don't (content gaps)
   - Backlink sources they have, you don't (link gaps)
   - Content types they use (video, tools, guides)

4. Benchmark:
   - If competitors DR=50, your DR=20 -> long-term link building
   - If competitors have 500 pages, you have 50 -> content production
   - If competitors have strong text optimization, you don't -> on-page work
```

### SEO roadmap template (first 6 months)

```
MONTH 1 - FOUNDATION:
[ ] Install analytics (GA4 + Yandex Metrica)
[ ] Submit to Google Search Console + Yandex Webmaster
[ ] Technical audit (full checklist)
[ ] Fix critical technical errors
[ ] Baseline position tracking setup

MONTH 2 - RESEARCH:
[ ] Build semantic core (full keyword research)
[ ] Cluster keywords and map to pages
[ ] Competitive analysis
[ ] Content plan creation
[ ] Structure improvements (if needed)

MONTH 3 - ON-PAGE:
[ ] Optimize meta tags (Title, Description, H1) for top pages
[ ] Create/optimize content for priority clusters
[ ] Internal linking optimization
[ ] Image optimization (alt, compression, WebP)

MONTH 4 - CONTENT:
[ ] Continue content creation per plan
[ ] Optimize secondary page clusters
[ ] Structured data implementation (Schema.org)
[ ] UX improvements based on behavioral metrics

MONTH 5-6 - AUTHORITY + EXPANSION:
[ ] Link building campaign launch
[ ] Guest posting / digital PR
[ ] Content updates for existing pages
[ ] Expand semantic core to new topics
[ ] Performance review and strategy adjustment
```

## Gotchas

- **"Do everything at once" fails** - trying to fix technical issues, create content, and build links simultaneously dilutes effort; sequential phases with clear priorities yield faster results
- **New sites need 3-6 months** before seeing meaningful organic traffic; setting unrealistic expectations with clients leads to churn
- **Quick wins exist** - fixing critical technical errors (broken canonicals, blocked crawling, missing meta tags) can produce ranking improvements in 2-4 weeks even before full strategy execution
- **Strategy must adapt** - a rigid 12-month plan that ignores algorithm updates, competitor moves, and performance data is useless; quarterly strategy reviews are mandatory
- **Budget allocation matters** - for competitive niches, splitting 50/50 between content and links is common; for low-competition niches, 80% content / 20% links works

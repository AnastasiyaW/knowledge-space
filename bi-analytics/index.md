## Key Formulas Quick Reference

```
Gross Profit = UA * (C1 * APC * AvP * Margin - CPA)
LTV = cumulative revenue per customer over lifetime
ROI = LTV / CAC
Retention = returning users / total cohort users
Data-ink ratio = data pixels / total pixels
```
